# Juan Villacorta Portfolio

Personal portfolio website built to showcase my projects, technical skills, and background as a Computer Science (Information Systems) student at California State University, Bakersfield.

This site is designed for recruiters, interviewers, and internship opportunities, with a clean multi-page layout and a professional presentation of my work.

## Features

- Modern multi-page portfolio design
- Responsive layout for desktop and mobile
- Dedicated pages for About, Skills, Projects, and Contact
- Downloadable résumé
- Project cards with technologies and GitHub links
- Professional dark-themed UI built for clarity and readability

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React

## Pages

- **Home** — introduction and landing page
- **About** — background, goals, and academic focus
- **Skills** — languages, frameworks, tools, and concepts
- **Projects** — selected technical projects and contributions
- **Contact** — ways to connect, plus résumé access

## Featured Project

### BlackAegis
Secure real-time messaging application available as both a web and desktop client.  
Features include end-to-end encrypted direct and group messaging, JWT authentication, Socket.IO real-time delivery, and a zero-trust design where message plaintext never reaches the server.

**My role:** Contributed primarily to backend architecture and implementation, and also to setup improvements, overall polish, and demo readiness.
